# WeJH-Taro

使用跨平台框架 Taro 构建的基于 Vue3 + TS 的小程序

## 介绍

 微精弘是是碎片化、一站式、一体化校园移动门户，适用于浙江工业大学学生，集课表、成绩、考试、空教室、一卡通、借阅等功能于一体，为工大学生的日常生活提供了便利
